## <a name="clean-up-resources"></a>Limpar os recursos

Para limpar todos os recursos usados durante este guia de início rápido, exclua o grupo de recursos:

```azurecli
az group delete \
    --name $AZ_RESOURCE_GROUP \
    --yes
```
